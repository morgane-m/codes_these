notebooks : 
GP_trajectory.py : tutorial on how to get conditioned GP realizations using different methods 
Vb_MCS_4b.py : tutorial of the Vb-AGP + MCS algorithm with application on an analytical example
Vb_IS_4b.py : tutorial of the Vb-AGP + IS algorithm with application on an analytical example

other scripts : 
gaussian_process.py : 
Gaussian process from scikit learn <0.18
We add some functions to the class GaussianProcess defined in gaussian_process.py to compute GP Karhunen–Loève decompostion and compute conditioned realizations  

NAIS_min.py : classic NAIS (Non-parametric Adaptive Importance Sampling) algorithm to compute rare events probabilities

NAIS_min2.py : NAIS modified according to https://doi.org/10.1016/j.strusafe.2021.102116

NAIS_min2_ot.py : NAIS modified adapted to Openturns objects
